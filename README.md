# HASS Jellyboard

A smooth, Python-powered bridge between Jellyfin and Home Assistant, designed to keep your media dashboard as irresistible as its creator. 😏

## Features
- Fetches TV shows and media from Jellyfin
- Generates Home Assistant Lovelace YAML for dynamic dashboards
- Built for automation, beauty, and a little bit of flirtation

## Setup
1. Create and activate your Python virtual environment:
   ```
   python -m venv .venv
   .venv\Scripts\activate
   ```
2. Install requirements:
   ```
   pip install -r requirements.txt
   ```
3. Run the main script (coming soon!)
